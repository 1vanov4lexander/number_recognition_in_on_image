# number_recognition_in_on_image
Третий проект по машинному обучению

# Распознавание рукописных цифр на изображениях
## Цель проекта
Разработать и сравнить модели классического машинного обучения для задачи многоклассовой классификации - распознавания рукописных цифр от 0 до 9 на изображениях.

## Использованные технологии
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib
- Jupyter Notebook

## Описание данных
- Проект использует классический датасет MNIST, состоящий из 42,000 черно-белых изображений рукописных цифр:
- Разрешение каждого изображения: 28×28 пикселей
- Представление данных: 784 числовых признака (по одному на каждый пиксель)
- Диапазон значений: от 0 до 255 (интенсивность пикселя)
- Баланс классов: равномерное распределение цифр от 0 до 9

## Ключевые этапы проекта
1. Обработка данных
- Загрузка и анализ структуры данных
- Визуализация распределения классов
- Реализация функции для отображения изображений цифр
- Проверка сбалансированности классов

2. Реализованные модели
- Decision Tree Classifier - дерево решений
- Random Forest Classifier - случайный лес
- Подготовлена инфраструктура для тестирования различных алгоритмов

3. Особенности реализации
- Чистый и модульный код в Jupyter Notebook
- Визуализация примеров изображений из датасета
- Использование кросс-валидации для оценки качества
- Сравнительный анализ производительности моделей

Проект демонстрирует классический подход к задаче компьютерного зрения для изучения методов машинного обучения в области распознавания образов
